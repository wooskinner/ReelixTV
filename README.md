# ReelixTV 🎬

A TV-optimized streaming app that works on Android TV, Fire TV, Apple TV, and Smart TVs.

## 🚀 Live Demo
- **TV App**: [https://wooskinner.github.io/ReelixTV/tv/](https://wooskinner.github.io/ReelixTV/tv/)

## 📱 Features
- ✅ TV-optimized interface for 10ft viewing
- ✅ Remote control support (arrow keys, Enter, Escape)
- ✅ Hero carousel with trending content
- ✅ Content rows (Trending, Popular, Action, Comedy, African Movies, TV Shows)
- ✅ Browse with genre filters
- ✅ Search functionality
- ✅ My List (watchlist)
- ✅ Firebase authentication
- ✅ Trial & subscription management
- ✅ PWA support

## 🛠️ Tech Stack
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **API**: TMDB (The Movie Database)
- **Auth**: Firebase Authentication
- **Database**: Firebase Firestore
- **Hosting**: GitHub Pages

```bash
# Clone the repository

cd ReelixTV

# Serve with Python
